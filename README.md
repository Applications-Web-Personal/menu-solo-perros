## Para el desarrollador

### Requisitos

- [Node.js](https://nodejs.org/) versión 18 o superior

### Correr localmente

```bash
npm install   # Solo la primera vez
npm run dev   # Abre http://localhost:4321
```

### Publicar en internet

**Vercel (más fácil):**

1. Ir a [vercel.com](https://vercel.com) → New Project → conectar repositorio → Deploy

**GitHub Pages:**

1. `git push` al repositorio
2. GitHub → Settings → Pages → Source: **GitHub Actions**

### Actualizar el menú

Edita `src/data/menu.ts` para cambiar precios, productos y fotos.
Ver `CLAUDE.md` para instrucciones detalladas.
